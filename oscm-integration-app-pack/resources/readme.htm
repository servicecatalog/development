Integration Package for Asynchronous Provisioning

Welcome to the integration package for asynchronous provisioning. This package helps you integrate your applications or other external systems, which need asynchronous provisioning, with Catalog Manager.

This package includes the Asynchronous Provisioning Platform (APP). APP is a framework which provides a provisioning service, an operation service, as well as functions, data persistence, and notification features which are always required for integrating applications with Catalog Manager in asynchronous mode. The package contains setup and configuration files for APP, an API to connect the applications with APP, samples, and the corresponding documentation.

Package Content

This package contains the following items:

  • APP Web service API. The API is used to implement service controllers managing the communication with the applications which are to be integrated with Catalog Manager.
    • oscm-app-extsvc.jar - a library containing the Web service API of APP.
    • API javadoc - the javadoc for the Web service API of APP.
    • API changes - Archive files, changes*.zip, where you can find detailed information on changes to the APP Web service API as compared to previous releases. Use the changes.html file in each archive for viewing the respective documentation.
  • Installation and configuration files:
    • oscm-app-install-pack.zip - ZIP file containing the installation scripts and configuration files for APP.

      When you extract the .zip file, you will find the oscm-app.ear file for APP in the domains/app_domain/applications subdirectory. This .ear file is to be deployed on an application server.

  • Samples:
    • GettingStarted - A simple implementation of a service controller which does not require a real application.
      oscm-app-sampleSRC.zip - ZIP file containing the sources of the sample service controller.
      oscm-app-sample.ear - EAR file containing the sample service controller ready to be deployed on an application server.
      Documentation - the description of the sample.
  • Documentation:
    • Step-by-step instruction of how to install APP and integrate applications with Catalog Manager for asynchronous provisioning.
